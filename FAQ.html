What are the requirements?

- dotNET Framework 4.7.2 and above

- VC Redist x86

- VC Redist x64

- Webclient Roblox without Beta App

Why does Windows Defender detect this as a virus?

Roblox script execution works by injecting a DLL into Roblox. The DLL can then do things to the client that it wasnt intended to do, such as executing Lua code.

DLL injections aren't just used for game cheats: they can be used maliciously. In this case, Cerus uses it for script execution, but your antivirus thinks that it can be used maliciously.

To resolve this, whitelist the entire Cerus folder in your antivirus.

How long do I have to wait for game updates?

We try to update to the latest version of Roblox as quickly as possible, which could be anywhere from a few hours to a day.
